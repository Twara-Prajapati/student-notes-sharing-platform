# Student Notes Sharing Platform

This is a simple full stack web application where students can upload, view, and download study notes.

## Features
- Upload study notes
- View available notes
- Download notes
- Simple and user friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## Purpose
This project was created as part of a full stack web development practical to demonstrate basic frontend and backend integration.
